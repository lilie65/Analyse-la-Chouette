Liste des erreurs non exhaustive

Vitesse du téléchargement du site

  1. Changer le format de certaines images vers des formats plus compressible (.bmp=> .jpg)
  2. Compresser les images avec un logiciel
  3. Redimensionner les images à la taille ou elles vont s'afficher (gain de temps pour le navigateur)
  4. Utiliser le cache navigateur pour le chargement plus rapide pour les prochaines visites
  5. Réduire le temps de lecture du fichier css et js

Un fichier robot.txt ainsi qu'un fichier sitemap permet d'optimiser son site et être compris par les moteurs de recherche

Réferencement

  1. Absence de fichiers sitemap et robots.txt
  2. Balise du code non optimisé(que des div)
  3. manque l'élément for dans la balise label
  4. Balise meta description vide
  5. le point dans la balise title ne constitue pas un descriptif
  6. Supprimer les mots clé caché et utilisé un référencement naturel
  7. la balise alt des images n'est pas renseigner de manière explicite

La balise alt est importante en accessibilité celle ci permet aux mal voyant d'accéder à l'image pour une retranscription vocal du texte
